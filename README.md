# hangmangame # 🎮 Hangman Game using Python

## 📌 Description
This is a text-based Hangman game where the player guesses a hidden word letter by letter.

## 🚀 Features
- Random word selection
- Limited attempts (6 tries)
- Displays progress after each guess

## 🛠️ Technologies Used
- Python
- Random module
- Loops and conditionals

## ▶️ How to Run
1. Open terminal
2. Run:
   python hangman.py
3. Guess the word!

## 📷 Sample Output
Word: _ _ _ _  
Guess a letter: a

- Working with lists and strings
- Game logic building
- Loop control
